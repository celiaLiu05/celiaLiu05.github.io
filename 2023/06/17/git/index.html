Git常用命令

工作区 —-git add—-> 暂存区 —-git commit—->本地库—-git push—->远程仓库

新建代码库

  • 初始化仓库 git init

  • 下载一个项目和它的整个代码历史 git clone [url]

  • 添加指定文件到暂存区 git add [file]

配置

  • 显示当前的Git配置git config --list

  • 编辑Git配置文件git config -e [--global]

  • 设置提交代码时的用户信息

    git config [--global] user.name "[name]" git config [--global] user.email "[email address]"

添加文件到暂存区

  • 添加当前目录的所有文件到暂存区git add .

代码提交

  • 将暂存区内容添加到仓库中 git commit -m "日志信息"

远程同步

  • 显示所有远程仓库 git remote -v
  • 本地仓库关联远程仓库并命名 git remote add [shortname] [url]
  • 上传本地指定分支到远程仓库git push [remote] [branch]
  • 取回远程仓库的变化，并与本地分支合并 git pull [remote] [branch]

分支

  • 查看当前分支 git branch -v
  • 创建新的分支且依然停留在当前分支 git branch [branch-name]
  • 新建一个分支，并切换到该分支 git checkout -b [branch]
  • 切换到指定分支 git checkout [branch-name]

查看信息

  • 显示有变更的文件 git status

  • 显示当前分支的最近几次提交 git reflog

  • 显示当前分支的版本历史 git log

撤销

  • 穿梭到某个版本 git reset --hard fcb7dbf(版本号)

合并分支

  • 把指定的分支合并到当前的分支上 git merge 分支名
